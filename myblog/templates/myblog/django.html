Django هو إطار عمل ويب مكتوب بلغة بايثون، يهدف إلى تسريع عملية تطوير التطبيقات مع التركيز على الأمان، وسهولة الاستخدام، والمرونة.

تثبيت Django:

  1. الخطوة 1: تأكد من تثبيت Python 3.x على جهازك.
  2. الخطوة 2: استخدم الأمر: 
    pip install django

لإنشاء مشروع Django جديد، استخدم الأمر: 

django-admin startproject myproject

لإنشاء تطبيق جديد داخل المشروع، استخدم الأمر: 

python manage.py startapp myapp

لإعداد قواعد البيانات، استخدم SQLite كقاعدة بيانات افتراضية. ثم نفذ الأوامر التالية:

python manage.py makemigrations
python manage.py migrate

النماذج تُستخدم لتمثيل البيانات في قاعدة البيانات:

from django.db import models

class Course(models.Model):
    title = models.CharField(max_length=100)
    description = models.TextField()

HTML (HyperText Markup Language) هي لغة ترميز تستخدم لإنشاء صفحات الويب. تعتمد HTML على عناصر وعلامات لتحديد هيكل ومحتوى الصفحة. إليك الأساسيات الكاملة لـ HTML:

1. هيكل الصفحة الأساسية
<!DOCTYPE html>
<html lang="ar">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>عنوان الصفحة</title>
</head>
<body>
    <h1>مرحبًا بك في HTML!</h1>
    <p>هذا هو نص تجريبي.</p>
</body>
</html>

تتكون HTML من عناصر، وهي تتكون من:

  • علامات بداية و علامات نهاية.
  • محتوى داخل العلامات.
<p>هذا نص داخل عنصر الفقرة.</p>

تستخدم لتنسيق النص كعناوين. هناك ستة مستويات من العناوين:

<h1>عنوان رئيسي</h1>
<h2>عنوان فرعي</h2>
<h3>عنوان فرعي أصغر</h3>

تستخدم لإنشاء فقرة نصية:

<p>هذه فقرة نصية.</p>

تستخدم لإنشاء روابط:

<a href="https://www.example.com">زر الزيارة</a>

تستخدم لإضافة الصور:

<img src="path/to/image.jpg" alt="وصف الصورة">

يمكنك إنشاء قوائم مرتبة وغير مرتبة:

قائمة غير مرتبة:
<ul>
    <li>العنصر الأول</li>
    <li>العنصر الثاني</li>
</ul>
قائمة مرتبة:
<ol>
    <li>العنصر الأول</li>
    <li>العنصر الثاني</li>
</ol>

لإنشاء جداول:

<table>
    <tr>
        <th>عنوان العمود 1</th>
        <th>عنوان العمود 2</th>
    </tr>
    <tr>
        <td>بيانات 1</td>
        <td>بيانات 2</td>
    </tr>
</table>

لإنشاء نماذج:

<form action="/submit" method="post">
    <label for="name">الاسم:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="إرسال">
</form>

لتنسيق الصفحات:

<style>
    body {
        background-color: #f8f9fa;
    }
    h1 {
        color: #333;
    }
</style>

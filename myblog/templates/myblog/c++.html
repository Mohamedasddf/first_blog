دورة C++ المجانية للمبتدئين

انطلق في رحلة تعلم البرمجة مع هذه الدورة المميزة.

ما هي C++؟: C++ هي لغة برمجة متعددة الأغراض، تُستخدم في تطوير البرمجيات، الألعاب، وتطبيقات الأنظمة.

مميزات C++:

  • أداء عالي وفعّال.
  • تسمح بالبرمجة الكائنية.
  • مجموعة واسعة من المكتبات المتاحة.

قم بتحميل C++ من أحد المواقع الرسمية مثل Microsoft Visual Studio.

تأكد من إعداد البيئة بشكل صحيح لتشغيل البرامج.

المتغيرات تُستخدم لتخزين البيانات:

أنواع البيانات:

  • int: أعداد صحيحة.
  • float: أعداد عشرية.
  • char: حرف.
  • string: نصوص.

تستخدم للتحكم في تدفق البرنامج:

int age = 18;
if (age >= 18) {
    std::cout << "مؤهل للتصويت";
} else {
    std::cout << "غير مؤهل للتصويت";
}

حلقة for:

for (int i = 0; i < 5; i++) {
    std::cout << i;
}

حلقة while:

int count = 0;
while (count < 5) {
    std::cout << count;
    count++;
}

تستخدم لتجميع الكود في وحدات قابلة لإعادة الاستخدام:

void greet(std::string name) {
    std::cout << "مرحبا، " << name << "!";
}

greet("Ali");
نهاية الدورة

C++ هي لغة برمجة قوية ومرنة. يمكنك استخدام الأساسيات التي تعلمتها لبناء تطبيقات صغيرة أو حتى مشاريع كبيرة. مع الممارسة، ستصبح أكثر إلمامًا بمفاهيمها المتقدمة. إذا كان لديك أي أسئلة أو تحتاج إلى توضيحات، فلا تتردد في طرحها!
